# 🚀 AI-Based Incident Analyzer

### AI-Augmented Security Operations Center (SOC) using Wazuh, Mistral, OpenCTI and PostgreSQL

> **An on-premises AI-powered SOC orchestration platform that automates the first level of security incident analysis while keeping the analyst in control.**

---

# 📌 Overview

**AI-Based Incident Analyzer** is an intelligent Security Operations Center (SOC) orchestration platform designed to reduce analyst workload by automatically processing, enriching and classifying security alerts generated by **Wazuh**.

Instead of manually investigating every alert, the platform leverages a **locally hosted Large Language Model (Mistral 7B via Ollama)** combined with **Threat Intelligence enrichment (OpenCTI)** to produce structured incident analyses, recommend response actions and map detected attacks to the **MITRE ATT&CK** framework.

The project follows a **Human-in-the-Loop** philosophy: the AI assists the analyst, but every defensive action requires human validation.

---

# 🎯 Objectives

* Automate first-level SOC alert analysis
* Reduce false positives through contextual enrichment
* Accelerate incident triage and classification
* Recommend appropriate response actions
* Map attacks to MITRE ATT&CK techniques
* Maintain complete incident traceability
* Preserve data confidentiality with a fully local deployment

## 🔍 Intelligent Alert Analysis

* Automatic JSON parsing
* Noise filtering

* Context-aware analysis

---

## 🤖 Local AI Copilot

Powered by:

* Mistral 7B Instruct
* Ollama REST API
* Custom SOC System Prompt
* Prompt Engineering
* Few-shot examples
* Structured JSON outputs

The model produces:

* Attack detection
* Severity score
* Threat classification
* Confidence score
* MITRE ATT&CK mapping
* Recommended response

---

## 🌍 Threat Intelligence

OpenCTI enriches alerts using:

* Malicious IP reputation
* Known Indicators of Compromise (IOC)
* Malware family information
* Threat confidence

---

## 🗄️ Incident Database

Every analysis is stored inside PostgreSQL including:

* Original alerts
* AI reasoning
* Severity
* Source IP
* Attack type
* Analyst decision
* Recommended action
* MITRE techniques
* Confidence score

---

## 👨‍💻 Human-in-the-Loop

Unlike fully autonomous SOAR platforms:
